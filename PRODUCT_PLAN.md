# SynClaw 产品计划

> 基于 Electron 桌面客户端 + OpenClaw Gateway 的 AI Agent 产品
>
> **技术架构**: Electron 28 + React 18 + TypeScript + Vite + Tailwind CSS + Zustand + OpenClaw Gateway (Node.js)

---

## 产品定位

| 维度 | 定义 |
|------|------|
| **产品名称** | SynClaw |
| **产品类型** | B2B/B2C 桌面客户端 - AI 助手本地文件操作平台 |
| **核心定位** | 让用户拥有自己的桌面 AI 助手，安全地操作用户本地文件系统 |
| **差异化优势** | 开箱即用（Electron 打包）、数据不落云端、炫酷 UI、华丽动画 |
| **架构模式** | SynClaw = OpenClaw 的桌面壳（Shell），所有 AI 能力来自内置 OpenClaw Gateway |

### 两种运行模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|

## 技术栈

### 客户端 (Electron)

| 层级 | 技术选型 | 理由 |
|------|----------|------|
| **桌面框架** | Electron ^28.0.0 | 跨平台桌面应用，成熟稳定 |
| **AI 后端** | OpenClaw Gateway | 所有 AI 能力来源，通过 GatewayClient 集成 |
| **UI 框架** | React ^18.2.0 | 组件化 UI |
| **语言** | TypeScript ^5.3.0 | 严格模式 |
| **样式** | Tailwind CSS ^3.3.0 | 原子化 CSS |
| **动画** | Framer Motion ^10.16.0 | 华丽流畅动画 |
| **状态管理** | Zustand ^4.4.0 | 轻量、简单（仅管理 UI 状态） |
| **构建工具** | Vite ^5.0.0 | 快速 HMR |
| **打包** | electron-builder ^24.9.0 | 成熟稳定 |

> **注意**: SynClaw 不使用 Go 后端。所有 AI 能力（推理、文件操作沙箱、记忆存储、技能执行）均委托给 OpenClaw Gateway。详见 `DEVELOPMENT_GUIDELINES.md`。

## UI 设计方向

### 设计风格

- **现代极简** - 大量留白、清晰层次
- **毛玻璃效果** - 使用 backdrop-blur 增加质感
- **微动画** - 按钮 hover、页面切换、加载状态
- **暗色优先** - 默认深色模式，护眼

### 关键界面

1. **主界面** - 左侧技能栏 + 右侧对话区 + 底部文件浏览器
2. **技能市场** - 卡片式布局、分类筛选、搜索
3. **文件浏览器** - 树形目录 + 文件预览 + 拖拽操作
4. **设置中心** - 分栏设置项、即时预览

### 视觉元素

- 渐变背景 (紫色/蓝色系)
- 圆角卡片 (12px-16px)
- 柔和阴影
- 图标: Lucide Icons
