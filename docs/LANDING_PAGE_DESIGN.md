## 2. 设计语言

### 2.1 视觉风格

**风格定位**：现代极简 + 科技感 + 暗色优先

参考：Cursor、Linear、Vercel、Raycast 的设计语言

```
关键词：极简 · 克制 · 科技感 · 暗色优雅 · 精准留白
```
